## 1. 프로젝트 소개

Travel-agency-website 프로젝트는 패키지 여행사 사이트 프로젝트입니다.  
남녀노소 모두가 좋아하는 여행지 유럽, 미국, 일본등 다양한 나라의 패키지 여행상품을 만들었습니다.    
각 페이지에는 패키지 설명, 가격 정보, 구매 버튼 등이 포함되어 있습니다.
